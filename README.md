# Inventory Management System (Java CLI)

This project is a command-line based Inventory Management System written in Java. It is designed as a university assignment to demonstrate fundamental object-oriented programming concepts, including abstraction, encapsulation, inheritance, and polymorphism. The system allows users to manage inventory through a text-based interface and persists data using a text file.

## Features

- Add new products (Clothing or Toy)
- Remove products by ID
- Restock existing products
- Purchase products with stock validation
- View all products in inventory
- File input/output using a plain text file
- Input validation and error handling

## Technologies Used

- Java 21
- NetBeans 23 IDE
- Standard Java libraries
